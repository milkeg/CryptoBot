# CryptoBot
Simple of a Crypto Bot connected to Gatecoin, Liqui and Bittrex.
It can be the baseground to move on to more complex logic.

How to use:
```
python3 trading_bot.py
```

Configuration file: strategy_config.py
You can put there the market you wanot to trade on and your API Keys
